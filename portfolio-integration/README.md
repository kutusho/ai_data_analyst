# Integración al Portafolio

Este directorio contiene los archivos listos para integrar el proyecto `AI Data Analyst Platform` en `ivandiazlab.com`.

## Archivos

- `projects-card-es.html`
- `projects-card-en.html`
- `detalle-ai-data-analyst.html`
- `detalle-ai-data-analyst-en.html`

## Dónde pegar cada archivo

### Tarjeta en español

Inserta el contenido de `projects-card-es.html` dentro del grid de proyectos en `projects.html`.

### Tarjeta en inglés

Inserta el contenido de `projects-card-en.html` dentro del grid de proyectos en `projects-en.html`.

### Página de detalle en español

Sube `detalle-ai-data-analyst.html` al mismo nivel donde ya existen páginas como:

- `detalle-sql.html`
- `detalle-churn.html`
- `detalle-flunex.html`

### Página de detalle en inglés

Sube `detalle-ai-data-analyst-en.html` al mismo nivel donde ya existen páginas como:

- `detalle-sql-en.html`
- `detalle-flunex-en.html`

## Recomendación

Coloca esta tarjeta al inicio del grid o dentro de los primeros tres proyectos. Es uno de los proyectos más fuertes de tu portafolio porque mezcla:

- producto completo
- backend + frontend
- analítica
- machine learning
- integración con LLM

## Ajuste opcional

La página actual de proyectos filtra por:

- `python`
- `sql`
- `tableau`
- `powerbi`

Esta integración usa `data-tags="python sql"` para no romper el filtro existente.
